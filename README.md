## ✨ 项目简介 (Introduction)

**RPGAme** 是一款正在开发中的 2D 像素风格横版动作游戏。玩家将通过极致的反应与策略，利用多种**御剑术**与**空间技能**击败敌人。核心体验专注于流畅的位移机制与充满变数的战斗系统。

## 🕹️ 操作指南 (Controls)

### 🏃‍♂️ 基础移动 (Movement)
| 按键 | 动作 | 描述 |
| :---: | :--- | :--- |
| **A / D** | **左右移动** | 基础跑动 |
| **Space** | **跳跃** | 探索高处必不可少 |
| **贴墙** | **攀爬/蹬墙跳** | 贴向墙壁时自动吸附，可进行进阶位移 |

### ⚔️ 战斗系统 (Combat)
| 按键 | 动作 | 描述 |
| :---: | :--- | :--- |
| **鼠标左键** | **三段攻击** | 连续点击触发平 A 三连击 |
| **Shift** | **瞬影冲刺** | 快速位移，并由**克隆体**对原地敌人发动攻击 |
| **Q** | **完美格挡** | 针对 **🔴红色标记** 的反击，可眩晕敌人并造成伤害 |
| **鼠标右键** | **万剑归宗** | 根据当前配置释放不同的御剑技能（普通/弹射/刺剑/旋转） |
| **R** | **黑洞/空间锁定** | 核心控制大招，结合 **QTE** 触发多重分身斩击 |
| **F** | **水晶技能** | 释放水晶，并通过与水晶互换位置或追踪敌人来爆炸造成伤害 |

## 🚀 版本更新日志 (Changelog)

### v0.0.4 - 🔮 水晶技能与冲刺升级
此版本增加了水晶技能、冲刺模式的改进，并进一步优化了格挡和黑洞技能，带来更加丰富的战斗体验。

* **[NEW] 水晶技能 (Crystal Skill)**：
  * **单次原地释放**：按下 `F` 键释放一颗水晶，玩家可以在水晶爆炸前与其互换位置。
  * **多重释放**：在规定时间内释放多个水晶，水晶追踪敌人并爆炸，造成范围伤害。
  * 每次释放水晶后会有冷却时间。

* **[NEW] 冲刺 (Dash)**：
  * **水晶追踪**：冲刺开始和结束时会分别释放一个水晶，水晶追踪并爆炸。
  * **克隆体攻击**：冲刺开始和结束时会分别生成一个克隆体并对敌人进行攻击。

* **[NEW] 格挡 (Parry)**：
  * **克隆体生成**：格挡成功后在敌人身旁生成克隆体，并有60%的几率继续生成克隆体。
  * **水晶爆炸**：格挡成功后生成水晶，向敌人移动并爆炸。

* **[NEW] 黑洞技能 (Black Hole)**：
  * **原有模式**：生成黑洞吸引敌人，时间暂停。
  * **新模式**：生成多个水晶飞向敌人并爆炸，造成范围伤害。

---

### v0.0.3 - 🌑 影落黑洞与万剑归宗
此版本极大地丰富了战斗的多样性，引入了进阶的御剑系统与复杂的空间控制技能。

* **[NEW] 进阶御剑系统 (Four Sword Arts)**：
  * **普通投出剑**：基础攻击，直线飞行并可召回。
  * **弹射剑 (Ricochet)**：剑会在多个敌人之间自动弹跳攻击，适合清场。
  * **刺剑 (Pierce)**：剑气凝聚，直接贯穿路径上的所有敌人。
  * **旋转剑 (Spinning AoE)**：剑在目标位置滞空旋转，形成持续伤害区域。

* **[NEW] 核心技：黑洞 (Black Hole)**：
  * **时空静止**：在敌人旁按下 `R`，玩家悬空并释放黑洞，黑洞内敌人时间暂停。
  * **QTE 机制**：玩家需按下敌人头顶显示的字母进行精准打击。
  * **分身追击**：完成 QTE 后再次按下 `R`，释放大量分身进行终结攻击。
