## ✨ 项目简介 (Introduction)

**RPGAme** 是一款正在开发中的 **2D 像素风横版动作 RPG / Roguelite Metroidvania**。  
玩家将通过极致的反应与策略，利用多种 **御剑术** 与 **空间技能** 在战斗中掌控节奏：

- 流畅的位移（冲刺 / 互换 / 黑洞浮空）
- 高节奏战斗（连击 / 反制 / QTE / 分身追击）
- 可扩展的属性与伤害系统（物理 / 魔法 / 元素异常）

---

## 🧩 核心特色 (Key Features)

### ⚔️ 战斗系统
- **三段普攻连击**：节奏感强，适配“打断 / 追击 / 躲避反打”
- **瞬影冲刺（Dash）**：快速位移，触发**水晶**与**克隆体**连携
- **完美格挡（Parry）**：针对“🔴红色标记”进行反击，触发眩晕与连锁生成
- **万剑归宗（右键）**：按配置释放不同御剑形态（普通 / 弹射 / 刺剑 / 旋转 AoE）
- **黑洞（R）**：核心控制大招，时间暂停 + QTE + 分身终结

### 🧠 属性与数值系统（v0.0.5 新增）
实体拥有更完整的 RPG 体系（玩家 / 敌人统一）：
- **基础属性**：力量、敏捷、智慧、活力  
- **战斗属性**：盔甲、闪避、暴击  
- **伤害类型**：物理伤害 / 魔法伤害  
- **元素异常**：燃烧、冰冻、电击（详见下文）

### ❤️ 血条 UI（v0.0.5 新增）
- 玩家与敌人均拥有 **血条 UI 系统**  
- 更直观地观察战斗反馈与 DOT（持续伤害）表现

## 📊 属性 & 伤害系统说明（v0.0.5）

> 这一版开始，游戏从“技能驱动”向“技能 + 数值驱动”演进。  
> 同一套 **CharacterStats** 体系统一服务玩家与敌人，便于后续扩展装备、词条、成长曲线。

### 🧬 基础属性 (Core Stats)
| 属性 | 作用（直接或间接） |
| :--- | :--- |
| **力量 (STR)** | 提升**物理输出**；影响**暴击相关**（暴击率/暴击增幅等） |
| **敏捷 (AGI)** | 影响**闪避率**；可能同时影响攻速/手感相关（可扩展） |
| **智慧 (INT)** | 提升**魔法输出**与元素相关强度（可扩展） |
| **活力 (VIT)** | 提升生命值上限与生存能力（可扩展） |

### 🛡️ 战斗属性 (Combat Stats)
| 属性 | 机制 |
| :--- | :--- |
| **盔甲 (Armor)** | **直接减少最终受到的伤害**（减伤型防御） |
| **闪避 (Evasion)** | 受敏捷影响，有一定几率**完全躲避一次攻击** |
| **暴击 (Critical)** | 受力量影响，有一定几率使本次物理攻击伤害提升 |

### 🗡️ 伤害类型 (Damage Types)
- **物理伤害（Physical）**：受力量等属性影响，走物理计算链
- **魔法伤害（Magical）**：受智慧等属性影响，走魔法计算链（并可能进入抗性/元素逻辑）

> 你可以把它理解为两条“并行伤害管线”：**DoPhysicalDamage / DoMagicalDamage**  
> 未来扩展（武器、装备、天赋、词条）会非常方便。

## 🔥❄️⚡ 元素异常系统（v0.0.5 新增）

### 🔥 燃烧 (Burn)
- 当武器附带燃烧属性并命中目标后：  
  目标在一段时间内持续受到燃烧伤害（DOT）

### ❄️ 冰冻 (Chill / Freeze)
- 当武器附带冰冻属性并命中目标后：  
  目标移动、跳跃、冲刺速度被减缓（减速效果）

### ⚡ 电击 (Shock)
- 当武器附带电击属性并命中目标后：  
  目标被标记为“电击状态”
- **再次攻击**该目标时：  
  从该目标位置生成一道闪电，劈向**最近的另一个敌人**并对其造成电击伤害/效果
