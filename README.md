## ✨ 项目简介 (Introduction)

**RPGAme** 是一款正在开发中的 2D 像素风格横版动作游戏。
玩家将通过极致的反应与策略，利用多种**御剑术**与**空间技能**击败敌人。核心体验专注于流畅的位移机制与充满变数的战斗系统。

## 🕹️ 操作指南 (Controls)

### 🏃‍♂️ 基础移动 (Movement)
| 按键 | 动作 | 描述 |
| :---: | :--- | :--- |
| **A / D** | **左右移动** | 基础跑动 |
| **Space** | **跳跃** | 探索高处必不可少 |
| **贴墙** | **攀爬/蹬墙跳** | 贴向墙壁时自动吸附，可进行进阶位移 |

### ⚔️ 战斗系统 (Combat)
| 按键 | 动作 | 描述 |
| :---: | :--- | :--- |
| **鼠标左键** | **三段攻击** | 连续点击触发平 A 三连击 |
| **Shift** | **瞬影冲刺** | 快速位移，并由**克隆体**对原地敌人发动攻击 |
| **Q** | **完美格挡** | 针对 **🔴红色标记** 的反击，可眩晕敌人并造成伤害 |
| **鼠标右键** | **万剑归宗** | 根据当前配置释放不同的御剑技能（普通/弹射/刺剑/旋转） |
| **R** | **黑洞/空间锁定** | 核心控制大招，结合 **QTE** 触发多重分身斩击 |

## 🚀 版本更新日志 (Changelog)

### v0.0.3 - 🌑 影落黑洞与万剑归宗 (Current)
此版本极大地丰富了战斗的多样性，引入了进阶的御剑系统与复杂的空间控制技能。

* **[NEW] 进阶御剑系统 (Four Sword Arts)**：
    * **普通投出剑**：基础攻击，直线飞行并可召回。
    * **弹射剑 (Ricochet)**：剑会在多个敌人之间自动弹跳攻击，适合清场。
    * **刺剑 (Pierce)**：剑气凝聚，直接贯穿路径上的所有敌人。
    * **旋转剑 (Spinning AoE)**：剑在目标位置滞空旋转，形成持续伤害区域。
* **[NEW] 核心技：黑洞 (Black Hole)**：
    * **时空静止**：在敌人旁按下 `R`，玩家悬空并释放黑洞，黑洞内敌人时间暂停。
    * **QTE 机制**：玩家需按下敌人头顶显示的字母进行精准打击。
    * **分身追击**：完成 QTE 后再次按下 `R`，释放大量分身进行终结攻击。

### v0.0.2 - 🗡️ 御剑术版本
* **[NEW] 投掷剑技能**：引入了蓄力瞄准、飞行穿透与召回机制。

### v0.0.1 - 🛡️ 影与盾版本
* **[NEW] 冲刺 (Dash)**：加入残影克隆体攻击机制。
* **[NEW] 格挡/招架 (Parry)**：加入基于敌方“危”字标记的反制系统。
