## ✨ 项目简介 (Introduction)

**RPGAme** 是一款正在开发中的 2D 像素风格横版动作游戏（Metroidvania）。
玩家将操控一名身手矫健的战士，探索未知的像素世界。核心体验专注于**流畅的移动机制**与**策略性的战斗系统**——既有拳拳到肉的三段击，也有考验反应的完美格挡，更有酷炫的远程御剑玩法。

## 🕹️ 操作指南 (Controls)

我们要的就是丝滑的操作体验！以下是目前的键位设定：

### 🏃‍♂️ 基础移动 (Movement)
| 按键 | 动作 | 描述 |
| :---: | :--- | :--- |
| **A / D** | **左右移动** | 基础跑动 |
| **Space** | **跳跃** | 探索高处必不可少 |
| **贴墙** | **趴在墙上** | 贴向墙壁时自动吸附，可进行爬墙或蹬墙跳 |

### ⚔️ 战斗系统 (Combat)
| 按键 | 动作 | 描述 |
| :---: | :--- | :--- |
| **鼠标左键** | **三段攻击** | 连续点击可触发平 A 三连击，拳拳到肉！ |
| **Shift** | **瞬影冲刺** | 快速向前方冲刺，**并留在一个克隆体对敌人发动一次攻击** |
| **Q** | **完美格挡** | 当敌人头顶出现 **🔴红色标记** 时按下，不仅能免伤，还能**眩晕敌人**并造成伤害 |
| **鼠标右键** | **飞剑 (投掷/召回)** | 长按瞄准 -> 松开飞出 -> 再次点击召回 (拥有回旋镖般的快乐) |

## 🚀 版本更新日志 (Changelog)

### v0.0.2 - 🗡️ 御剑术版本 (Current)
在这个版本中，我们极大地丰富了远程攻击手段，主角现在像个真正的剑仙了！
* **[NEW] 投掷剑技能**：
    * **蓄力瞄准**：按住 `鼠标右键` 进入瞄准模式。
    * **飞剑出击**：松开右键，剑会沿着预定轨迹高速飞出，穿透敌人。
    * **剑刃召回**：当剑在空中时，再次按下 `鼠标右键`，剑会飞回角色手中（并在返回路径上造成二次伤害）。

### v0.0.1 - 🛡️ 影与盾版本
加入了核心的身法和反击机制，让战斗更具策略性。
* **[NEW] 冲刺 (Dash)**：
    * 按下 `Shift` 进行快速位移。
    * **特性**：冲刺不再只是躲避，由于速度过快，会在原地生成一个**残影克隆体**，克隆体会自动对范围内的敌人进行一次攻击。
* **[NEW] 格挡/招架 (Parry)**：
    * 加入了“危”字机制！当敌人头顶出现 **红色惊叹号/标记** 时，是进攻也是防守的最佳时机。
    * 按下 `Q` 键触发格挡，成功后敌人将进入 **僵直（晕眩）状态** 并直接扣除血量。
